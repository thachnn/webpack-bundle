# Purpose

Bundle NPM packages using `webpack` and provide them :)

## Usage

```json
{
  "dependencies": {
    "glob": "thachnn/webpack-bundle#glob-7.2.3"
  },
  "resolutions": {
    "**/ajv": "thachnn/webpack-bundle#ajv-6.12.6"
  }
}
```
